Synopsis

 

This document shows you how you can manage your SAAS (Online) licenses for TraceCloud.

Use Case

 

After trying TraceCloud for some time, a user may decide that TraceCloud provides value to their organization and may decide to buy a license for using TraceCloud. This document shows the user the options he / she has and how one can go about purchasing, granting and managing TraceCloud licenses.

Licensing Models

 

TraceCloud comes as both an OnPremises solution and as a SAAS (OnLine) solution. This document covers only the SAAS (OnLine) licensing aspect.

The SAAS model supports the following license types

User Licenses:

  • Read Write User license. This license permits the user to create unlimited projects / requirements in the system
  • Read Only User license. This license permits the user to access unlimited projects / requirements in the system. With this license a user cannot create / modify information in TraceCloud.
  • At the time of this documentation, a Read Write license is priced at $30 per month and a Read Only license is priced at $25 per month. This pricing can change over time.

Project Licenses:

  • There are situations where you can have a large number of users who may need to access your project and you dont want to buy licenses for all of them.
  • Also, there are time times when the budgeting / spending needs to be done on a per project basis.
  • In this scenario, you can simply buy a project license and then unlimited number of users can access / modify this project. You wont need to buy individual licenses.
  • At the time of this documentation, this is the pricing for project licenses. Please note that this pricing can change over time.
    • $350 per month for a project with less than 5,000 requirements
    • $700 per month for a project with more than 5,000 but less than 10,000 requirements
    • $1000 per month for a project with more than 10,000 requirements

Notes:

  • With the SAAS model you pay for only for what you use. Since the billing is on a per user-month or per project-month basis you can quickly ramp up / down your usage, without having to make a large upfront investment.
  • The security to your project is controlled at your project level, by your project administrator. Just because you are paying for other users licenses, does not mean they can access your projects. Their access to your projects (or conversely your access to their projects) is driven by security settings at a project level. It has nothing to do with licensing.

Purchasing Licenses

 

To purchase SAAS TraceCloud licenses, log into your project and click on License Management tab. Depending on how many Read Write, Read only user licenses, you may want to enter the appropriate fields. If you however decide to buy project licenses, simply select the projects from the Project Licenses drop down. Remember to click the Update Licenses button. Depending on your needs you may want to mix and match the User and Project licenses.

Once again, to summarize: a user license lets the user access / creates unlimited projects and requirements. However each user needs a user license. A Project license on the other hand, can let unlimited users access this project, but you will need to buy a project license for every project.

After you purchase a user license, you will need to grant the licenses to the appropriate users.

Granting User Licenses

 

After you purchase user licenses, you will need to grant them to your users. You can grant them, by entering a comma separated list of email addresses in the Grant Read / Write and Grant Read Only licenses boxes. You will see this option in the License Management tab.

 

Revoking Licenses

 

When you want to revoke user or project licenses, simply log into TraceCloud and go the License Management tab. At the bottom you will see a section for revoking licenses. Select the Project or User license you want to revoke and it will return the license back to your license pool.

 

Billing

At the end of every month, TraceCloud billing department looks at all the user and project licenses held by you and send you an invoice. You can choose to pay by Credit Card or by mailing a check or by a bank transfer to a TraceCloud account. If you choose to use the Credit Card option (Visa, MasterCard, Amex etc...), please do the following.

Log into TraceCloud and click on the Account Profile tab and enter your credit card information.
